# Product Overview

**smallCount** is a cross-platform personal finance management application built with React Native and Expo. It helps users track income, expenses, and analyze spending habits with AI-powered insights.

## Core Features

- **Transaction Management**: Record income, expenses, and transfers with detailed categorization
- **AI Assistant**: Intelligent financial analysis and personalized recommendations using natural language interaction
- **Multi-Account Support**: Manage multiple accounts (cash, bank cards, credit cards, digital wallets)
- **Data Import/Export**: Import bills from Alipay/WeChat, export data to Excel format
- **Smart Analytics**: Visual charts and statistics with trend analysis
- **Flexible Tagging**: Custom income/expense categories with colors and icons
- **Theme Support**: Light/dark mode with multiple color themes (default, blue, purple, orange)

## Target Users

Individual users seeking comprehensive personal finance tracking with AI-powered insights for better financial decision-making.

## Key Value Propositions

1. **Privacy-First**: All data stored locally in SQLite database
2. **AI-Powered**: Intelligent spending analysis and budget recommendations
3. **Cross-Platform**: Runs on iOS, Android, and Web
4. **Offline-First**: No cloud dependency, complete local operation
5. **Modern UI**: Beautiful, responsive design with smooth animations
